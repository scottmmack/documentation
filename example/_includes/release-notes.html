Release Notes - v0.1

First Design

This is the very first release. It includes a modern yet serious design. Feel free to adjust anything as you'd like but don't blame me if you mess things up!

Modular, kinda

The real content is included in the snippets in includes. Most everything else is icing on the cake. Each file defines which layout it should use and which snippets are to be included. One can also include content and it will be injected in the middle of the layout.

Includes Bootstrap

A static version of bootstrap is included with both CSS and javascript. The screenshot carousel, for instance is pretty much plug-n-play bootstrap

Includes Font-Awesome

Font-Awesome removes the need for sprites and icons as it contains hunderds of icon in a font. Just use an "i" tag with the proper classes.
